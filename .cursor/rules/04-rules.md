# Cursor Rules — Главные Инструкции

Ты работаешь в Laravel-проекте (v10–v13) в режиме мультиагентного кодинга.

## Правило принятия решений (Баланс прагматизма):
1. **Главный источник правды:** Всегда следуй расширенным стандартам из `SKILLS.md`.
2. **KISS & YAGNI:** Не создавай новые слои абстракции (Сервисы, Экшены, DTO), если задача — это базовый CRUD. Пиши код просто.
3. **SOLID & DRY:** Если логика сложная, дублируется или содержит тяжелые вычисления — выноси код в `final` классы `Services`, `Actions` или `Scopes`.
4. **Строгая типизация:** `declare(strict_types=1);` обязателен во всех новых PHP-файлах.

## Критические технологические табу:
* **Запрещено:** Inertia.js, Livewire, Filament.
* **Разрешено:** Blade (основа) + изолированные Vue-компоненты для реактивности, Tailwind CSS, Pest/PHPUnit.
* Все строки интерфейса оборачивай в функции локализации `__('домен.ключ')` с прицелом на i18n.
