# Стандарты Кода и Стилизация

## PHP & Laravel
* Обязательно `declare(strict_types=1);` в начале PHP-файлов.
* Строгая типизация аргументов и возвращаемых значений (return types).
* Классы `Services`, `Actions`, `DTO` обязаны быть объявлены как `final`.
* Используй фичи PHP 8.2+: readonly-свойства, constructor property promotion, выражения `match`, Enums.
* Контроллеры одного действия (`__invoke`) для сложных, объемных эндпоинтов.
* Атрибуты моделей приводятся к Enums через конфигурацию `casts`.

## Фронтенд & Локализация
* Основа — Blade-компоненты (`<x-layout>`).
* Vue.js — только независимые компоненты, монтируемые через Vite в Blade-разметку. Передавай данные через пропсы: `:data="{{ json_encode($data) }}"`.
* В верстке используй утилитарные классы Tailwind CSS. Разделение через SCSS допускается только для кастомных `@keyframes`.
* Полная поддержка i18n через языковые файлы Laravel. Никакого хардкода строк в разметке.
