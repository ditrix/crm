1. добавить панел с элементами:
слева переключатель "показать архивные" в виде стилизованого чекбокса  
справа input поле поиск + символ очистки этого поля. поиск ведется по всем видимым данным и по мере совпадения количество отобржаемых строк уменьшается.

4 в записи клиента добавить загрузку фотографии (логотипа) клиента (отдельным полем)
если изображение не загружено показывать изображение no_image_icon.svg
