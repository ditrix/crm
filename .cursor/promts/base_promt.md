## УДАЛЕНИЕ ДАННЫХ

все записи из бизнес-данных имеют **soft_delete** формат
при изменении бизнес данных в запись добавляется информация какой пользователь системы (имя) выполнил изменения + updated_at в формате "dd.mm.yy HH:MM"
удаленные записи в таблице скрыты, есть переключатель "показать архивные", тогда в таблице видны все записи при этом удаленные немного затемнены и для них доступен только режим просмотра  

## Загрузка
в бизнесс данные можно загружать документы и изображения изображение. для загруженые бизнес файлов можно скачивать или просматривать в новой вкладке праузера с названиями с которыми они были загружены. для клиента можно загружать изображение-фотографию в качестве userpic
для клиента и сделки есть поле "Комментарий" который используется только как вспомогательный комментарий для пользователя и может быть пустым
Для загрузки доступен режим drug drop
